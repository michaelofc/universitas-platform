# MÓDULO 2 — EVENT DISPATCHER  
**Processamento de Eventos com Retry e Background Jobs**

## Visão Geral

**Objetivo:** Transformar eventos recebidos (Módulo 1) em ações práticas, aprendendo a tomar decisões baseadas em dados e executar workflows condicionais com resiliência.

**Produto Final:** Event Router Pro - sistema que lê eventos, classifica por tipo e executa ações específicas com retry automático.
